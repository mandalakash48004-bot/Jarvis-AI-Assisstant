# Jarvis AI Assistant

A Python-based AI voice assistant capable of understanding voice commands, answering queries, performing web searches, opening websites and applications, and retrieving real-time information through API integration.

## Features
- Voice command recognition
- AI-powered question answering
- Web search automation
- Website and application launching
- Real-time information retrieval
- Text-to-speech responses

## Technologies Used
- Python
- Speech Recognition
- Text-to-Speech (TTS)
- API Integration
- Automation Libraries
